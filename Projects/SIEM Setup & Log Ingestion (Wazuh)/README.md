Project 1: SIEM Setup & Log Ingestion using Wazuh

Objective: To design and deploy a working SIEM environment that collects and analyzes security logs from Windows and Linux systems.

Tools Used:

> Wazuh SIEM

> VirtualBox

> Windows 10 VM

> Linux VM

Implementation Steps:

> Installed Wazuh server (manager, indexer, dashboard)

> Configured network communication between VMs

> Deployed Wazuh agent on Windows endpoint

> Deployed Wazuh agent on Linux endpoint

> Validated agent connectivity and data ingestion

> Generated real log events and monitored in dashboard

Results:

> Successfully received logs from multiple endpoints

> Detected authentication failures and process execution events

> Achieved centralized monitoring through SIEM dashboard

Skills Demonstrated:

> SIEM deployment

> Agent configuration

> Log ingestion

> Event monitoring

> SOC environment setup
